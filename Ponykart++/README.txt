# Build and run
- Install MSVC 2013 or gcc >= 4.8.1 as the default compiler
- Install the dependencies (use the same compiler for all the dependencies and the project)
- Compile the project, a VS2013 and Code::Blocks project files are provided for Windows and Linux
- Copy the media/ folder in the build/ folder
- Run the game in the build folder, with the required .dllS / .soS

# Dependencies : 
- Ogre and OIS 1.8.1
- Bullet2 (Dynamics, Collision, LinearMath) >= 2.81-r2613
- Boost >= 1.55.0
- Lua 5.1
- irrKlang
